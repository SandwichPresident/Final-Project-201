# User Testing Documentation

## Overview
The application was tested by three users outside the development team to evaluate usability, error handling, and clarity.

## Tester Feedback

### Tester 1
- Login errors were confusing when credentials were incorrect.

### Tester 2
- Checkout should not allow an empty cart.

### Tester 3
- Wanted confirmation that saved data persists.

## Outcome
Feedback was analyzed and informed improvements related to validation, error handling, and file persistence.
